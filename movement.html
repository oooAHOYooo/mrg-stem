Scratch Movement + Makey Makey
Grades 3–5

Learning Target

I can make a character move left, right, and jump in Scratch and explain what a forever loop does.

Success Criteria

  • Sprite moves left (change x by -10)
  • Sprite moves right (change x by 10)
  • Sprite jumps (repeat up / repeat down)
  • forever loop runs ongoing actions (e.g., gravity)

Materials

  • Scratch on Chromebooks
  • Makey Makey + alligator clips + foil pads
  • Printed Scratch cards
  • Projector / whiteboard

How Scratch Movement Works

  • Left/Right: change x. Right is positive; left is negative.
  • Up/Down: change y. Up is positive; down is negative.
  • Jump: increase y a few times, then decrease to land.
  • Gravity: a small negative y in a forever loop keeps pulling down.
  • Input: when [key] pressed blocks run movement on key press.

Keyboard Code (start here)

when [right arrow v] key pressed
  change x by (10)

when [left arrow v] key pressed
  change x by (-10)

when [space v] key pressed
  repeat (10)
    change y by (5)
  repeat (10)
    change y by (-5)

Add Gravity with forever

forever
  change y by (-3) // try -2 to -5

Makey Makey Setup

  1. Clip EARTH to a foil wrist band or ground pad.
  2. Connect ←, →, and SPACE to foil pads labeled LEFT / RIGHT / JUMP.
  3. Touch pads to control the same Scratch code.

Forever Loop — Visual

Without a forever loop, gravity and input would run once and stop.

Differentiation

  • New coders: Only left/right + simple jump.
  • Ready: Add gravity, ground color check, or animation.
  • Finish early: Add coin/obstacle + score.

Table of Contents — Step by Step

  1. Open Scratch and choose a sprite and backdrop.
  2. Add left/right movement with change x by on key press.
  3. Create a jump with two repeats: up then down.
  4. Add gravity using a small negative y in a forever loop.
  5. Connect Makey Makey: map ←, →, and SPACE to pads.
  6. Test, tweak speeds, and share what your loop does.
